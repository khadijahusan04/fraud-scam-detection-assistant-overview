# Fraud/Scam Detection Assistant

Fraud/Scam Detection Assistant is a full-stack web application that helps users review suspicious messages, links, emails, screenshots, and reports for possible fraud or scam activity.

This public repository is a project overview only. The source code is private.

## Live Demo

View the live project here:
https://scam-fraud-assistant.vercel.app

## Overview

Fraud/Scam Detection Assistant was built to help users identify warning signs in suspicious digital content. Users can submit text, links, emails, or screenshots, and the system provides a risk-based analysis to help them make safer decisions.

The project combines full-stack development, data tracking, dashboard reporting, case management workflows, and AI-assisted classification. It was designed to support suspicious activity review, fraud awareness, and analytics-based decision-making.

## Problem

Online scams and phishing attempts are becoming harder to detect because suspicious messages often look realistic. Users may receive fake links, urgent requests, impersonation attempts, fake verification messages, or suspicious screenshots.

This project was created to give users a simple tool for reviewing suspicious content and understanding the potential warning signs before taking action.
